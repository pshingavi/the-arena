# Lenny's Data ZIP

This ZIP contains the starter pack with a limited number of newsletter posts and podcast transcripts.

## What is included

- Newsletter posts: **10**
- Podcast transcripts: **50**
- `01-start-here/index.json` is the source of truth for this ZIP and uses the bundled file paths
- Free subscribers can connect to the starter MCP, and paid subscribers can unlock the full archive at [lennysdata.com](https://www.lennysdata.com)

## ZIP layout

- `01-start-here/`: `README.md`, `LICENSE.md`, and `index.json`
- `02-newsletters/`: curated starter newsletter markdown posts
- `03-podcasts/`: starter podcast markdown transcripts
